Korcen은 korean(한국어) censorship(검열)의 줄임말입니다

누구나 쉽게 검열이 가능합니다

한국 비속어가 주이지만 다양한 언어를 지원합니다

[PY 버전 바로가기](https://github.com/KR-korcen/korcen)

[서포트 디스코드](https://discord.gg/wyTU3ZQBPE)

## Installation
>PyPI
```sh
$ pip install korcen
```

>NPM
```sh
$ npm i korcen
```
# Example
버전에 따라 달라질 수 있습니다 

>Python
```py
from korcen import korcen

text = input()
print(korcen.check(text))
```
